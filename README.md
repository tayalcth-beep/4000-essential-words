# 🎯 4000 Essential English Words - 智慧行動測驗系統

一個專為《4000 Essential English Words》打造的輕量化、無後端、跨平台「智慧單字測驗網頁」。支援手機極致流暢操作，並內建生字簿與手動美式發音功能，隨時隨地高效背單字！

---

## ✨ 核心特色

* **📱 極致行動端優化**：防誤觸藍底設計（No-Select），高頻點擊選項不卡頓；同時保留例句選取功能，方便查字。
* **🔊 點擊手動發音**：採用瀏覽器內建 Web Speech API，作答完畢後可手動點擊「播放發音」，背單字不劇透、不干擾。
* **❌ 核心生字簿**：錯字自動儲存於瀏覽器本機快取（LocalStorage），並依據錯誤次數降序排列，精準打擊弱點。
* **📊 學習儀表板**：自動記錄歷史總測驗場次與綜合答對率，學習反饋不中斷。
* **🔒 防呆鎖定**：篩選單一 Unit 時自動鎖定 20 題，完美契合原書結構。

---

## 🛠️ 本地開發與打包步驟

本專案採用「範本注入技術」，僅需在本機執行 Python 腳本，即可將幾千個單字打包成單一 `index.html` 網頁。

### 1. 準備檔案
請確保你的本機資料夾內包含以下三個檔案：
* `4000_Essential_English_Words_revised.json` (單字資料庫)
* `quiz_template.html` (智慧網頁範本)
* `build_quiz.py` (自動打包腳本)

### 2. 執行打包
在該資料夾下打開終端機（Terminal），執行以下指令：
```bash
python build_quiz.py
